== Description ==

= Build your pricing table. =

This plugin gives you the possibility to build your pricing table or your table comparison, select a pre-built layout and display it thanks to a shortcode.

You’ll be able to dynamically add a row or a column to your pricing table and insert your data.

The goal of the plugin is to let you create a pricing table, switch easily between several layouts and not wasting time with complexe shortcode.

= How to use our plugin ? =

* You just have to download your plugin and upload it to your "wp-content/plugins" folder.
* Activate your plugin on your Dashboard and you will see a Pricing Table tabs.
* At this point, you just have to create a new Pricing Table by completing the fields.
* Once created, go back to the list of your Pricing Tables and you will see a Shortcode.
* All you have to do, is put the Shortcode wherever you want to get your Pricing Table.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/table-shortcode` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Usage ==
1. Simply use the new tab in the admin panel named Pricing Table.
2. Create your first item.
3. Go back to the item's list and grab the shortcode.
4. Insert the shortcode wherever you want.
